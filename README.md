# CarZone – Demo Car Accessories Website

This is a simple multi-page demo website for a fictional car accessories brand called **CarZone**.

## Pages

- `index.html` – Home page with a hero section and 4–5 featured products with images.
- `about.html` – About page describing the story and values behind CarZone.
- `contact.html` – Contact page with a demo contact form (no backend).
- `checkout.html` – Mock checkout page with a shipping & payment form and pre-filled order summary.

## Tech Stack

- Plain **HTML**, **CSS**, and a small amount of inline **JavaScript** for form interactions and the footer year.
- No build tools or external dependencies are required.

## How to Run

Simply open any of the `.html` files in your browser:

1. Locate this folder on your machine.
2. Double-click `index.html` to open it in your default browser.
3. Use the navigation bar to move between pages.
